API Reference

AutoScrape provides a REST API for ZIP-code-targeted vehicle listing scraping across Cars.com and CarGurus. All endpoints return JSON. All write operations require authentication.

Base URL
http://localhost:8000
or your deployed host
Format
JSON request / response
multipart/form-data for CSV upload
Interactive
localhost:8000/docs
Swagger UI (FastAPI)

Authentication

🔑All endpoints require an X-API-Key header. Set your key in .env as API_KEY=your-key-here.
bash 
# Pass the header on every request
curl -H "X-API-Key: your-key-here" http://localhost:8000/listings/

Missing or invalid key returns 403 Forbidden. The API key is validated in app/core/security.py and can be rotated by updating the API_KEY env var and restarting the server.

Error Codes

CodeMeaningCommon Cause
400Bad RequestInvalid adapter name, malformed ZIP, missing required field
403ForbiddenMissing or invalid X-API-Key header
404Not FoundJob ID does not exist

POST/scrape/

Enqueue a scrape job. Returns immediately with a job_id. The job runs asynchronously via Celery. Poll GET /scrape/jobs/{job_id} for progress.

Request Body

FieldTypeRequiredDescription
zip_codesstring[]requiredList of 5-digit US ZIP codes to search
radius_milesintegeroptionalSearch radius in miles. Default: 50
adaptersstring[]optionalAdapter names. Default: ["cars_com","cargurus"]
max_pages_per_adapterintegeroptionalPages to scrape per adapter per ZIP. Default: 3
respect_robotsbooleanoptionalHonour robots.txt. Default: true
headlessbooleanoptionalRun Playwright headless. Default: true
bash 
curl -X POST http://localhost:8000/scrape/ \
  -H "X-API-Key: your-key" \
  -H "Content-Type: application/json" \
  -d '{
    "zip_codes": ["75201", "75034"],
    "radius_miles": 50,
    "adapters": ["cars_com", "cargurus"],
    "max_pages_per_adapter": 3
  }'

Response 202 Accepted

json 
{
  "job_id": "a3f2c8d1-1234-5678-abcd-ef0123456789",
  "status": "pending",
  "message": "Job enqueued for 2 ZIP(s) × 2 adapter(s)."
}

GET/listings/

Browse all stored listings with optional filters.

ParamTypeDescription
zip_codestringFilter by exact ZIP code
makestringCase-insensitive partial match on make
sourcestringFilter by adapter: cars_com or cargurus
limitintegerMax results (≤500). Default: 100
offsetintegerPagination offset. Default: 0
bash 
curl "http://localhost:8000/listings/?zip_code=75201&make=Toyota&limit=50" \
  -H "X-API-Key: your-key"

GET/listings/export/csv

Stream filtered listings as a CSV file download. Accepts the same filter params as GET /listings/. No pagination limit — streams up to 10,000 rows.

bash 
curl "http://localhost:8000/listings/export/csv?zip_code=75201" \
  -H "X-API-Key: your-key" -o listings.csv

GET/listings/export/json

Same as CSV export but streams a JSON array. Up to 10,000 records per request.

POST/admin/import/csv

Upload a CSV file with a zip_code column (or a plain text file with one ZIP per line). AutoScrape deduplicates the ZIPs and enqueues a single scrape job.

Form FieldTypeDescription
fileFileCSV or TXT file
radius_milesintegerDefault: 50
adaptersstringComma-separated. Default: "cars_com,cargurus"
max_pages_per_adapterintegerDefault: 3
bash 
curl -X POST http://localhost:8000/admin/import/csv \
  -H "X-API-Key: your-key" \
  -F "file=@zip_codes.csv" \
  -F "radius_miles=50" \
  -F "adapters=cars_com,cargurus"

POST/admin/import/zips

Same as CSV import but accepts a JSON body — useful for programmatic ingestion.

bash 
curl -X POST http://localhost:8000/admin/import/zips \
  -H "X-API-Key: your-key" \
  -H "Content-Type: application/json" \
  -d '{"zip_codes":["75201","75034","78201"],"radius_miles":50}'

Adapters

Adapters are the site-specific scraping modules. Each adapter extends BaseAdapter and is registered in app/adapters/registry.py. To add a new site, create a new file and add it to the registry.

NameSiteModeCrawl Delay
cars_comcars.comHTTPX + BeautifulSoup2.0s / page
carguruscargurus.comPlaywright (Chromium)3.0s / page

Selectors drift as sites update their DOM. Run make test-live weekly to detect breakage. When a test returns 0 results (not an error), update the CSS selectors in the adapter file.
